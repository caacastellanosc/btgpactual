# Manejo de PQR

Se requiere implementar un sistema que permita a los clientes de la compañía realizar las siguientes operaciones:

1. Registrar una solicitud de tipo “Petición”

2. Registrar una solicitud de tipo “Queja”

3. Registrar una solicitud de tipo “Reclamo”:

4. Consultar las PQR que el usuario haya registrado previamente.

Reglas de negocio:

• Toda petición, queja o reclamo debe generar un número de radicado único

• Solo se puede registrar un reclamo sobre una petición o queja realizada hace más de 5 días, o que tenga respuesta del área administrativa y cuya respuesta no sea de agrado al usuario.

• Cada PQR registrada por el usuario final, debe poder ser consultada en el sistema, y debe mostrar el contenido de la PRQ, así como la respuesta del área administrativa (si tuviese)

• Cada Reclamo registrado por el usuario final, debe poder ser consultado en el sistema, y adicionalmente debe poder visualizarse el contenido de la petición o queja que dio origen al reclamo, así como la respuesta a esa petición o queja.
